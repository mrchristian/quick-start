Step 1: Make a Repository

A repository is the data storage location of your outputted publication.

The reposoitories use Git1 technology which allows for versioning of your publication with crytographic IDs.

We save to GitHub and GitLab (Perkel  2016) – including GitLab.com or another self-hosted instance of the GitLab Community Edition which is open-source software.

1 Git is open-source software that both GitHub and GitLab are built on – think of it as a time machine for code and all that could do.
